# Queue System

## Overview
Queue System 是一個商店排隊管理平台，提供商店管理目前正在排隊的顧客以及顧客目前的服務進度。
商家在每次申請帳號後(開店)，可以使用此系統 24 小時，接著系統就會自動將帳號停止（關店）。帳號停止後，商家會收到一份 csv 報表，紀錄當日所有顧客的詳細資訊，方便商家做商業分析。隔天商家要開店並使用此系統，只要再重新申請帳號即可使用。
